--- Desafio Projeto Hospedagem

Este projeto é parte do desafio da DIO para a trilha .NET. O objetivo é simular um sistema de hospedagem com reserva de suítes e cálculo de valores.

--- Funcionalidades
- Cadastro de hóspedes
- Cadastro de suíte
- Validação de capacidade
- Cálculo de valor da diária com desconto para reservas acima de 10 dias

--- Regras
- A suíte deve comportar todos os hóspedes
- Reservas com 10 ou mais dias recebem 10% de desconto
